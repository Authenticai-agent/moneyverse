# LLM Boundaries

## Allowed future uses

- Rephrase approved educational explanations
- Generate schema-constrained story variations
- Draft parent summaries
- Suggest practice-question variations
- Assist moderation with human review

## Prohibited uses

- Move money
- Connect accounts
- Initiate payments
- Award XP or tokens
- Change permissions
- Publish content automatically
- Send private messages to children
- Recommend individual securities
- Retrieve unrestricted database records
- Execute code or shell commands
- Make irreversible administrative decisions

## Required controls

- Disabled provider and mock provider
- Strict runtime schemas
- Minimized context
- No secrets in prompts
- Tool allowlists
- Deterministic post-policy checks
- Cost and rate limits
- Timeouts and circuit breakers
- Human approval for consequential outputs
- Full kill switch
