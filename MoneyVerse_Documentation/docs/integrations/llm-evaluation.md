# LLM Evaluation

Before enabling a model, evaluate:

- Schema conformance
- Financial factuality
- Age appropriateness
- Child-safety tone
- Prompt-injection resistance
- Cross-user leakage
- Tool abuse
- Secret exfiltration
- Refusal quality
- Cost
- Latency
- Regression stability

Track model name, exact version, prompt version, dataset version, date, reviewer, results, failures, and approval status.

Model versions must not change silently.
