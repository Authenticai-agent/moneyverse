# AI Data Policy

## Never send

- Passwords or PINs
- JWTs or refresh tokens
- API keys
- Plaid access tokens
- Card or bank credentials
- Social Security numbers
- Full legal child identity
- Precise location
- Unrestricted database records
- Private support or security investigations

## Allowed only after review

- Pseudonymous age band
- Approved lesson ID
- Minimized progress metrics
- Curated scenario inputs
- Redacted adult feedback

## Provider requirements

- Contract and privacy review
- Training-use controls
- Retention controls
- Region assessment
- Deletion process
- Security documentation
- Incident notification
