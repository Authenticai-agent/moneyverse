# Product Edge Cases

- Two guardians share one family.
- A guardian loses access or custody.
- A child changes age band.
- A child belongs to multiple classrooms.
- A teacher leaves a school.
- A lesson is updated while in progress.
- A reward request times out after database commit.
- Two refresh requests arrive simultaneously.
- A family requests deletion while a school has an educational record obligation.
- A parent withdraws consent for an optional feature.
- A school removes a student but the family keeps personal progress.
- Currency support expands beyond USD.
- A family loses access to the registered email.
- A child nickname contains unsafe or identifying information.
- An integration is disabled while jobs are queued.
- A parent disputes a virtual-balance adjustment.
- A content item is retired after assignment.
- A child uses accessibility accommodations that affect timing-based anti-abuse signals.

For each implemented feature, convert applicable edge cases into tests or documented operational decisions.
