# Launch Checklist

## Security

- [ ] Authentication review complete
- [ ] Authorization matrix implemented
- [ ] RLS tested
- [ ] JWT negative tests pass
- [ ] Refresh replay test passes
- [ ] Rate limits verified
- [ ] CSP and security headers verified
- [ ] Secret scan clean
- [ ] Dependency scan reviewed
- [ ] Penetration test complete
- [ ] Admin MFA enforced
- [ ] Backups and restore tested
- [ ] Incident runbooks reviewed

## Privacy and child safety

- [ ] Data inventory complete
- [ ] Retention automation verified
- [ ] Deletion flow tested
- [ ] Consent versions recorded
- [ ] No public child profiles
- [ ] No child-to-stranger messaging
- [ ] No targeted advertising
- [ ] Share cards contain no identifiers
- [ ] Child-facing copy reviewed
- [ ] Legal/privacy review complete

## Accessibility

- [ ] WCAG 2.2 AA audit
- [ ] Keyboard navigation
- [ ] Screen-reader testing
- [ ] Contrast verification
- [ ] Reduced-motion behavior
- [ ] Text scaling
- [ ] Accessible charts and game controls

## Product and operations

- [ ] MVP acceptance criteria complete
- [ ] Error monitoring configured
- [ ] Alert ownership assigned
- [ ] Support workflow ready
- [ ] Status page or communication plan ready
- [ ] Production secrets rotated
- [ ] Production data seed contains no real personal data

## App stores, if applicable

- [ ] Age rating reviewed
- [ ] Privacy labels completed
- [ ] Child-app questionnaires reviewed
- [ ] COPPA-related answers reviewed by counsel
- [ ] Data deletion URL active
- [ ] Privacy policy and terms links active
- [ ] Screenshots contain no personal data

## Feature flags

- [ ] LLM disabled unless separately approved
- [ ] Plaid disabled unless separately approved
- [ ] Payments disabled unless separately approved
- [ ] Public social features disabled
