# Account Takeover Runbook

1. Confirm indicators and affected account scope.
2. Revoke all sessions and refresh-token families.
3. Require password reset and MFA review.
4. Review role, family, consent, export, and integration changes.
5. Preserve redacted evidence.
6. Notify the account owner through a trusted channel.
7. Restore unauthorized changes where safe.
8. Record root cause and preventive actions.
