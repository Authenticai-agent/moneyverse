# Incident Response

## Lifecycle

1. Detect
2. Triage
3. Contain
4. Preserve evidence
5. Eradicate
6. Recover
7. Assess notification duties
8. Review and improve

## Priority incidents

- Account takeover
- JWT signing-key exposure
- Database credential leak
- Child-data exposure
- Admin compromise
- Malicious dependency
- Plaid token compromise
- Payment-provider compromise
- Ransomware
- Data corruption
- Suspicious bulk export
- Production misconfiguration

## Requirements

- Named incident lead
- Severity criteria
- Secure communication channel
- Evidence handling
- Credential rotation plan
- User impact assessment
- Legal escalation
- Recovery validation
- Post-incident action tracking
