# Data Dictionary

**Family** — Private tenant connecting guardians and child profiles.

**Guardian** — Adult with an authorized relationship to a family and one of the approved family roles.

**Child profile** — Minimized child learning identity; not automatically an adult user account.

**Simulated cash** — Educational value with no cash value.

**Learning XP** — Non-purchasable measure of educational progress.

**Cosmetic token** — Non-random, non-transferable game currency with no cash value.

**Allowance** — Parent-entered or scheduled simulated amount; not a bank transfer or payroll obligation.

**Mission** — Defined educational or parent-approved activity with server-controlled completion rules.

**Lesson version** — Immutable published snapshot used to interpret historical progress.

**Mastery** — Evidence-based proficiency across one or more attempts.

**Integration connection** — Adult-authorized external provider relationship, isolated from core learning data.

**Audit event** — Append-oriented record of a security, permission, administrative, consent, or critical financial-style action.
