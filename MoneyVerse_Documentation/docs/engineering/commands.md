# Safe Commands

## Approved routine commands

- Install with frozen lockfile
- Lint
- Type-check
- Unit test
- Integration test
- E2E test
- Security test
- Build
- Migration status
- Local development migration
- Local seed using synthetic data

## Explicit approval required

- Production migration
- Database reset
- Destructive seed
- Force push
- History rewrite
- Secret rotation
- Production deployment
- Deleting storage objects
- Disabling security controls
- Adding an external registry
- Running remote scripts

Never pipe remote content directly into a shell.
