# MoneyVerse Documentation Repository

This repository is the operating system for building MoneyVerse safely and consistently with Windsurf, Cursor, Claude Code, or another coding agent.

## Start here

Every coding task must begin by reading, in order:

1. `AGENTS.md`
2. `windsurfrules.md`
3. `current-task.md`
4. `tasks.md`
5. `architecture.md`
6. `schema.md`
7. `decisions.md`
8. The relevant document under `/docs`

## Product status

- Product: MoneyVerse
- Stage: Secure foundation
- LLM: Disabled by default
- Plaid: Disabled by default
- Payments: Disabled by default
- Public social features: Prohibited
- Child-to-stranger messaging: Prohibited

## Documentation rules

- Root files are authoritative and intentionally concise.
- Detailed policies live under `/docs`.
- The implementation must match documentation.
- Material implementation changes require a documentation update in the same pull request.
- Legal compliance requires qualified legal review; code and documentation alone do not establish compliance.
