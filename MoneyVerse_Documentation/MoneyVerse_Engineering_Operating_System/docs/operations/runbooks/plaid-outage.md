# Runbook: Plaid Outage

1. Confirm incident and scope.
2. Assign incident lead and severity.
3. Contain the affected path.
4. Preserve redacted evidence.
5. Rotate or revoke credentials if relevant.
6. Validate child-data and financial-data impact.
7. Recover through tested procedure.
8. Communicate through approved channels.
9. Add regression tests and action items.
